# Orbit Looper ベータテスト — テスター向けガイド

Orbit Looper（AI Personal OS）の TestFlight ベータにご参加いただきありがとうございます。

## インストール方法

1. iPhone に **TestFlight** アプリを App Store からインストール（未インストールの場合）
2. 開発者から届いた **招待メール**を開く
3. **View in TestFlight**（TestFlight で表示）をタップ
4. **インストール**または**更新**をタップ

招待リンクが届かない場合は、開発者に Apple ID のメールアドレスを共有してください。

## このベータについて

- **TestFlight ベータ版**です。不具合や未完成の機能がある場合があります
- データは **端末内のみ** に保存されます（クラウド同期なし）
- 機種変更前は **設定 → データ → エクスポート** でバックアップを取ってください

## 試してほしいこと

### 1. はじめの 1 日（Today）

- アプリを開き、**Today** タブで今日の予定を確認
- タスクを完了したり、セッションを開始してみてください
- 「今日は PUSH / NORMAL 」など **DayType** の表示に違和感がないか

### 2. フォーカス（Focus）

- セッションから **フォーカスモード** に入る
- カウントダウンと **端末を置いた状態** の検知が自然か

### 3. ふりかえり（Reflection）

- 1 日の終わりに **ふりかえり** を記入（表示された場合）
- 気分・エネルギーなどが自然に記録されるか

### 4. カレンダー

- 予定の **追加・編集**
- タイムライン上での **時間変更**

### 5. 設定

- **Orbit Looper AI** セクションでプラン表示を確認
- **データ → エクスポート** が動作するか

## フィードバックの送り方

1. アプリの **設定** タブを開く
2. 下の方にある **ベータ** セクションを確認
3. **フィードバックを送る** をタップ

または、開発者から指定された方法（メール・GitHub など）で送ってください。

### 報告に含めてほしい情報

- **バージョン**（設定 → ベータ に表示）
- **端末**（例: iPhone 15, iOS 18.x）
- **再現手順**（何をしたら問題が起きたか）
- **スクリーンショット**（あれば）

## よくある質問

**Q. Expo Go とは違いますか？**  
A. はい。TestFlight 版は Orbit Looper 専用のビルドで、通知など Expo Go では使えない機能も含みます。

**Q. データは消えますか？**  
A. ベータ更新で通常は残りますが、重大な変更時は開発者から事前連絡します。こまめにエクスポートをおすすめします。

**Q. 有料プランは必要ですか？**  
A. いいえ。無料プランでも Today の自動配置などコア機能は動作します。Orbit Looper AI（コーチ等）は有料プラン向けです。
